## Despliegue en Netlify

1. Sube esta carpeta a un repositorio de GitHub
2. Conecta el repositorio en Netlify
3. Netlify detectará automáticamente `index.html` como punto de entrada
4. Configura tu dominio personalizado en Netlify > Domain settings
